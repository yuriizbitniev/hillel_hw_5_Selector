Селектори

Основні види селекторів css

  • Універсальний селектор Вибирає усі елементи. За бажанням, він може бути обмежений певним простором імен або ставитись до всього простору імен
  • Селектори за типом елемента Цей базовий селектор вибирає тип елементів, до яких застосовуватиметься правило.
  • Селектори за класом Цей базовий селектор вибирає елементи, ґрунтуючись на значенні їх атрибуту. class
  • Селектори з ідентифікатора Цей базовий селектор вибирає елементи, ґрунтуючись на значенні їхнього id атрибута. Не забувайте, що ідентифікатор повинен бути унікальним, тобто використовуватися тільки для одного елемента HTML-документі
  • Селектори з атрибутуЦей селектор вибирає всі елементи, що мають цей атрибут або атрибут з певним значенням
